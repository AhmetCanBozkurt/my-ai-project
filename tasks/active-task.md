# Aktif Görev

Bu dosyaya AI Agent'a yaptırmak istediğiniz görevi yazın.

## Örnek Görevler:

### Örnek 1: Yeni bir API endpoint ekle
```
Projeye yeni bir REST API endpoint ekle:
- GET /api/health endpoint'i oluştur
- Endpoint "status": "ok" döndürmeli
- Express.js kullan
```

### Örnek 2: Yeni bir özellik ekle
```
Kullanıcı profil sayfası oluştur:
- React component'i oluştur
- Kullanıcı bilgilerini göster
- Modern ve responsive tasarım
```

### Örnek 3: Bug fix
```
Login sayfasındaki şifre validasyon hatasını düzelt:
- Minimum 8 karakter kontrolü ekle
- Büyük/küçük harf ve rakam zorunluluğu
```   
  
---

## Şu Anki Görev:

Basit bir "Hello World" API endpoint'i oluştur:
- Express.js kullanarak bir Node.js server oluştur
- GET /api/hello endpoint'i ekle
- Endpoint "Hello, World from AI Agent!" mesajı döndürmeli
- server.js dosyası olarak kaydet
- package.json'a "start" script'i ekle
